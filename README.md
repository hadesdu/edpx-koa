edpx-koa
========

edpx-koa是基于es6 Generator开发的webserver。实现的功能同edp-webserver。使用前需要安装nodejs 0.11.x以上版本。

##安装使用

+ 安装nodejs 0.11.x版本

+ 安装edp

```
$ npm i -g edp
```

+ 修改~/.bashrc

```
alias node='/usr/bin/env node --harmony'
```

+ 生效bashrc配置

```
$ source ~/.bashrc
```

+ 启动webserver

```
$ edp koa start
```
